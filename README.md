## 📖 Project Overview

## 🛠 Tools & Technologies Used

- PySpark

---

## 📊 Dataset Details
- Total Records: 100,000
- Columns:
  - Transaction_ID
  - Category
  - Amount
- Categories:
  - Electronics
  - Clothing
  - Groceries
  - Books

---

## 🔍 Analysis Performed
- Counted total records
- Grouped data by category
- Calculated average transaction amount per category
- Generated summary statistics (mean, min, max, stddev)
